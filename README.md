# 3aeShop - Premium Gaming Laptop Store

A full-stack e-commerce application for premium gaming laptops built with React, Vite, Express, and Tailwind CSS.

## Features

- **Premium UI**: Modern dark-themed gaming aesthetic.
- **Product Catalog**: Explore the latest gaming laptops with detailed specs.
- **Shopping Cart**: Add to cart, update quantities, and manage items.
- **QR Payment**: Vietnamese Bank QR payment integration (VietinBank).
- **Admin Panel**: Manage products (Add, Edit, Delete) and track orders.
- **Authentication**: Simple login/register system for users and admins.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4, Framer Motion (Motion).
- **Backend**: Express.js (Node.js).
- **Icons**: Lucide React.
- **Animations**: Motion (fka Framer Motion).

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd 3aeshop
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

**Development Mode:**
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

**Production Mode:**
```bash
npm run build
npm start
```

# 3aeShop - Cửa hàng Laptop Gaming Cao Cấp

Ứng dụng thương mại điện tử full-stack dành cho laptop gaming, xây dựng bằng React, Vite, Express và Tailwind CSS.

## Tính năng

- **Giao diện Cao cấp**: Phong cách gaming hiện đại, chuyên nghiệp.
- **Danh mục Sản phẩm**: Khám phá các dòng laptop gaming mới nhất.
- **Giỏ hàng**: Thêm vào giỏ, cập nhật số lượng và quản lý sản phẩm.
- **Thanh toán QR**: Tích hợp quét mã QR ngân hàng (VietinBank).
- **Trang Admin**: Quản lý sản phẩm (Thêm, Sửa, Xóa) và theo dõi đơn hàng.
- **Đăng nhập/Đăng ký**: Hệ thống tài khoản cho khách hàng và quản trị viên.

## Hướng dẫn Cài đặt

1. Cài đặt các gói phụ thuộc: `npm install`
2. Chạy chế độ phát triển: `npm run dev`
3. Xây dựng bản chính thức: `npm run build` ➔ `npm start`
